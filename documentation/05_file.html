Functions

Visual.customHint.clearTimer()

Clears the timer which is set to hide the CustomHint after some interval.

Visual.customHint.adjustSizeNPosition(xPos, yPos)

Adjusts the size and the position of the hint div according to the data inside it and the given pop coordinates.

parameters

type name description
integer xPos (optional) The x coordinate on the page where the hint box will be popped.
integer yPos (optional) The y coordinate on the page where the hint box will be popped.

Visual.customHint.pop(message, xPos, yPos)

Pops up the hint div with a fade effect at the given coordinates with the given message. Automatically called by the generated mousemove event handler but might be called manually to show hints.

parameters

type name description
String message (optional) The message which will appear in the hint. The string here is assigned to the innerHTML of the customHint.
integer xPos The x coordinate on the page where the hint box will be popped.
integer yPos The y coordinate on the page where the hint box will be popped.

Visual.customHint.hide()

Hides the customHint with a fade effect and clears the hide timer.

Visual.customHint.onmouseout()

Sets the hide timer of customHint when the cursor moves away from it.
This event assignment should not be overridden but developers can use Visual.customHint.onmouseout() to manually set the timer for hiding.
