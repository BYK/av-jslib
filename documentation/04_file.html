Constructors

Visual.sliders()

Introduces a namespace that holds the sliders' functions under the Visual namespace.

parameters

type name description
Boolean config.effects (optional) Set to true to turn on the css hover and mousedown effects. Default is true.
integer config.heightDifference (optional) Automatically calculated height difference between the slider and the neighbour divs. You may override it manually.

objects

type name description
HTMLDivElementObject activeSlider The property stores the active slider's object referance whenuser is silding a slider. You should not change its value but you may use its value.
