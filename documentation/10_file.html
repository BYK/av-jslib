Constructors

AJAX.dynamicLanguage()

Represents a namespace under AJAX, AJAX.dynamicLanguage, for the new functions and global parameters of those functions.

parameters

type name description
Boolean config.retryOnError Set true for recursive retries if a language file request fails.
Function(String) config.loadingNotifier The callback function which is called when the language file is loading. The language code of the loaded language is passed as the only parameter.

objects

type name description
XMLHttpRequestObject _loader The XMLHttpRequestObject which is used to load the language asynchronously
XMLElementCollectionObject _strings The collection which holds the retrieved language strings

AJAX.dynamicLanguage.fillTexts(container)

This function scans the sub elements of the given object for langText defined elements and then sets their innerHTML's to the appropriate text, retrieved dynamically from the XML language file.

parameters

type name description
HTMLElementObject, String container (optional) The id or the object reference of the element-to-be-scanned for dynamic language text associations.
If not spesified, the function automatically scans the whole document(usually this is useful).

AJAX.dynamicLanguage.load(code)

Loads and fills the language texts whose code is given as the parameter.
'lang/'+code+'.xml' is the path for the language file. You may change this pattern @ line 100

parameters

type name description
String code The language code which is used to load the language file

returns

type description
XMLHttpRequestObject The created XMLHttpRequest object which is used to load the language file.
