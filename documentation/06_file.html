integer Visual.infoBox.config.timeout

The time interval before the infoBox is automatically hid in milliseconds. Default is 3000.

Functions

Visual.infoBox.clearTimer()

Clears the timer which is set to hide the infoBox after some interval.

Visual.infoBox.show(message, showImmediately, timeout)

Shows the info box with the given message.

parameters

type name description
String message (optional) The message which will appear in the info box. The string here is assigned to the innerHTML of the infoBox object.
If not set, the innerHTML of the infoBox is left the same.
Boolean showImmediately (optional) If set, the infoBox appears instantly, without the fade effect.
Integer timeout (optional) The message spesific timeout in milliseconds. If not given, config.timeout is used.

Visual.infoBox.hide()

Hides the infoBox with a fade effect and clears the hide timer.
