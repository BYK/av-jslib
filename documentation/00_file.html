AJAX&XML Library

Constructors

AJAX()

Represents a namespace, AJAX, for the new functions and global parameters of those functions.

parameters

type name description
String config.noAjax (optional) The error message which user will see if his/her browser does not support AJAX. If you want to disable this warning, just set this to false.
String config.loadImgPath (optional) The "loading" gif's path, which might be used in various places.
String config.loadingText (optional) The "loading" message which will be placed into a dynamically filled container while the content is being loaded. If you want to disable this text, just set this to false.
String config.pageLeaveWarning (optional) The warning message which will be displayed to user if (s)he tries to leave the page while an AJAX request is loading. If you want to disable this warning, just set this to false.

objects

type name description
integer activeRequestCount The current active requests number. Changing this value is discouraged.

AJAX.makeGetRequest(adress, changeFunction)

Creates a new XMLHttpRequest object which connects to the adress, which includes the URI encoded and merged GET parameters. Assignes changeFunction to the newly created XMLHttpRequest object's onreadystatechange event. Frees the XMLHttpRequest object automatically after completing the call.

parameters

type name description
String adress The adress of the page which will be connected. Should include the URI encoded GET parameters.
Function(XMLHttpRequestObject) changeFunction (optional) The function which will be assigned to the newly created XMLHttpRequest object's onreadystatechange event.

returns

type description
XMLHttpRequestObject

AJAX.makePostRequest(adress, parameters, changeFunction)

Creates a new XMLHttpRequest object which connects to the adress, which includes the URI encoded and merged POST parameters. Assignes changeFunction to the newly created XMLHttpRequest object's onreadystatechange event. Frees the XMLHttpRequest object automatically after completing the call.

parameters

type name description
String adress The adress of the page which will be connected. Should NOT include any parameters.
String parameters The URI encoded and merged POST parameters for the HTTP request.
Function(XMLHttpRequestObject) changeFunction (optional) The function which will be assigned to the newly created XMLHttpRequest object's onreadystatechange event.

returns

type description
XMLHttpRequestObject

AJAX.makeRequest(method, adress, parameters, completedFunction, loadingFunction)

Takes "GET" or "POST" as method parameter and then according to this, creates a SELF-MANAGED XMLHttpRequest object using internal makeGetRequest or makePostRequest according to the method parameter. Developers are strongly adviced to use THIS function instead of the above POST and GET specific functions.

parameters

type name description
String method Should be either POST or GET according to the type of the HTTP request.
String adress The adress of the page which will be connected. Should NOT include any parameters.
String, Object parameters The parameters which are either URI encoded and merged or given in the JSON format
Function(XMLHttpRequestObject) completedFunction (optional) The function which will be called when the HTTP call is completed. (readyState==4)
Function(XMLHttpRequestObject) loadingFunction (optional) The function which will be called EVERYTIME when an onreadystatechange event is occured with a readyState different than 4. Might be called several times before the call is completed.

returns

type description
XMLHttpRequestObject The newly created XMLHttpRequest object for this specific AJAX call.

AJAX.loadContent(theURL, element, completedFunction, loadingFunction)

Loads a dynamic content to the given container element. If the config["loadingText"] is defined, target container element's innerHTML is filled with global variable config["loadingText"] while the content is loading.

parameters

type name description
String theURL The URL of the content which will be loaded dynamically into the given container.
String, Object element The container element itself or its id, which the dynamic content will be loaded into.
Function(Object, String) completedFunction (optional) The function which will be called when the loading of the content is finished. It is called with the target container element and the URL as parameters.
Function(Object, String) loadingFunction (optional) The function which will be called EVERYTIME when an onreadystatechange event is occured with a readyState different than 4 while loading the dynamic content. It is called with the target container element and the URL as parameters.

AJAX.loadResource(theURL, type, resourceId, forceRefresh)

Loads an external style-sheet or Javascript file on-demand. Removes the old node if a resourceId is given.

parameters

type name description
String theURL The address of the resource-to-be-loaded.
String type (optional) The type of the resource. Should be either js or css. The default value is js.
String resourceId (optional) The id which will be assigned to newly created DOM node. If not given, no id is assigned to the created node.
Boolean forceRefresh (optional) Addes a "?time" value at the end of the file URL to force the browser reload the file and not to use cache.

returns

type description
HTMLElementObject The newly added script or link DOM node.

AJAX.XML.getValue(mainItem, nodeName, defaultVal)

Tries to extract the node value whose name is given with nodeName and is contained by mainItem node. Returns the defaultVal if any error occurs.

parameters

type name description
Object mainItem The main node item which holds the sub nodes and their values.
String nodeName Name of the sub node whose value will be extracted from the mainItem.
String defaultVal (optional) The default value which will be returned if the sub node whose name is given in nodeName is not found.

returns

type description
String The value of the node whose name is given with nodeName and which is contained by mainItem node.

AJAX.XML.setValue(mainItem, nodeName, val)

Tries to set the node value whose name is given with nodeName and is contained by mainItem node. Returns false if any error occurs.

parameters

type name description
Object mainItem The main node item which holds the sub nodes and their values.
String nodeName Name of the sub node whose value will be set.
String val The new value of the sub node whose name is given in nodeName.

returns

type description
String The value set by the function is returned. If an error occures, the function returns false.

AJAX.XML.toArray(collection)

Converts an element/node collection, which acts as an array usually, to an actual array and returns it, which allows developers to use array-spesific functions.

parameters

type name description
HTMLCollectionObject collection The collection which will be converted to array.

returns

type description
HTMLElementObject[] The array version of the given collection.
